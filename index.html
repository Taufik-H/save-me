Game Rules

  1. Klik tombol START di bawah untuk memulai permainan.
  2. Anda diberikan 5 kali kesempatan untuk menjawab pertanyaan.
  3. Jika Anda BENAR, maka game akan berlanjut ke soal selanjutnya.
  4. Jika Anda SALAH setelah 5 kali kesempatan menjawab, maka Anda akan diberitahu
    jawaban dari pertanyaan tersebut kemudian diarahkan ke soal selanjutnya.
  5. Diakhir game akan terdapat hasil dari apa yang kamu kerjakan.
  6. Enjoy the game, let's jumped to the questions.
